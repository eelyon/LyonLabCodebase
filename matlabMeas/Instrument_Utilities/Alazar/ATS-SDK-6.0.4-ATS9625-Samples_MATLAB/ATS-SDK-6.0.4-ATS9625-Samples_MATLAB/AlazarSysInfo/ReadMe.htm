Introduction

This folder contains a sample program that displays information about AlazarTech digitizer board found in a computer. It demonstrates how to get a handle to a board, and use the handle to get information about the board.

 

This sample is part of the AlazarTech SDK, and is documented in the “AlazarTech SDK Programmer’s Guide”.

System requirements

MATLAB® version 7.2.0.232 (R2006a) or later is required to load the script. At least one AlazarTech digitizer board, and AlazarTech software driver version 5.7.20 or later is required to run the script.

Running a sample program

·        Start MATLAB.

·        Change to the directory containing AlazarSysInfo.m.

·        Type “AlazarSysInfo” at the command prompt to run the program.
