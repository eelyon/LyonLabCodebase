Introduction

The ATS-SDK includes MATLAB sample programs that demonstrate how to configure and acquire data from AlazarTech digitizer boards.  These sample programs are organized into folders under the “Samples_MATLAB” folder, with one folder per board model.

 

Please refer to the AlazarTech SDK Guide for more information about using these sample programs.
